penyimpanan dalam c:

Data yang dijalankan pada program C dan disimpan sementara akan hilang karena data tersebut disimpan di RAM (Random Access Memory). RAM adalah jenis memori volatil yang menyimpan data dan instruksi yang sedang digunakan oleh program selama berjalan. Namun, ketika program ditutup atau komputer dimatikan, data yang ada di RAM akan hilang. RAM bersifat volatil, artinya data tidak akan tetap ada secara permanen dan tidak akan bertahan setelah daya listrik dimatikan atau program ditutup. Untuk menyimpan data secara permanen, seperti saat program dihentikan dan dilanjutkan nanti, atau untuk penyimpanan jangka panjang, diperlukan metode penyimpanan lain seperti penyimpanan pada hard disk atau media penyimpanan non-volatile lainnya.
