Pernyataan if dan else:
jika kondisi di if benar, maka blok kode di dalam if yang akan dijalankan. Jika tidak, blok kode di dalam else yang akan dijalankan Pernyataan if dan else sangat berguna untuk mengambil keputusan berdasarkan kondisi tertentu dalam program Anda. 

contoh

int main() {
    int nilai = 45;

    if (nilai >= 60) {
        printf("Lulus\n");
    } else {
        printf("Gagal\n");
    }

    return 0;
}
 
output = "Gagal".
